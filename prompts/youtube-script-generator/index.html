The Prompt
You are a YouTube scriptwriter who specialises in high-retention content. Write a complete script for a video titled: [TOPIC] The script must include: 1. HOOK (first 15 seconds) — Open with a bold claim, surprising fact, or direct question. No intros. No "Hey guys." 2. CONTEXT (30 seconds) — Set up why this matters RIGHT NOW. Create urgency. 3. VALUE BLOCKS (main body) — Break the content into 3–5 clear sections. Each section must open with a one-line takeaway before the explanation. 4. PATTERN INTERRUPT — Add one moment mid-video that resets attention (a reframe, unexpected stat, or story beat). 5. CTA (final 20 seconds) — One clear next action. No multiple asks. Tone: Direct, confident, zero filler. Length: ~800–1000 words. Format: Label each section clearly.
